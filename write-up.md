# Reflection Tree Design

This project is a deterministic reflection system based on three psychological axes:

1. Locus of Control (Victim vs Victor)
2. Contribution vs Entitlement
3. Self vs Others

The system uses structured questions with fixed options and follows a predefined decision tree.

Each user choice leads to a specific path, ensuring consistency and no randomness.

I used AI tools for brainstorming ideas, but I manually designed the logic, structure, and flow of the tree.

The aim is to guide users toward self-awareness without judgment.

With more time, I would expand the tree and improve the depth of reflections.
