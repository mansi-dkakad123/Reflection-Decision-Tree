# Deterministic Reflection Agent

This project is a structured reflection tool built using a decision tree.

## Features
- Fully deterministic (no randomness)
- No AI used at runtime
- Fixed options for all questions
- Structured reflection flow

## Project Structure
/tree/
  reflection-tree.json → Main logic
  tree-diagram.png → Flow diagram

write-up.md → Design explanation

## How it works
User selects options → system follows tree → reflections are shown.
